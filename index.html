¿Qué es la Web?

La Web es un sistema que permite acceder a información a través de internet. Funciona mediante navegadores que solicitan datos a servidores y muestran páginas con contenido como texto, imágenes y videos.

Tecnologías principales

  • HTML - Estructura del contenido
  • CSS - Diseño y estilos
  • JavaScript - Interactividad
